## ðŸš§ Getting Started

### Prerequisites
- Node.js & npm
- Java (JDK 11+)
- Angular CLI
- Maven

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
ng serve
```

By default, the frontend runs on `http://localhost:4200` and the backend on `http://localhost:8080`.
